# DQN-keras
Deep Q Learning agent using Keras for CartPole simulation using GYM.

![GitHub Logo](/images/demo.gif)


## Quickstart
  ```
  pip3 install -r requirements.txt
  python3 main.py
  ```

## Libraries used
- OpenAI gym
- numpy
- Keras
